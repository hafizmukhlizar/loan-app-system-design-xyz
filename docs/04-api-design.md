# API Documentation

| No | Screen     | Method & End Point            | Fungsi                                      | Header                      | Request Body                                                     | Response                                                     | Login Status |
| :- | :--------: | :---------------------------- | :------------------------------------------ | :-------------------------- | :--------------------------------------------------------------- | :----------------------------------------------------------- | :----------: |
| 1  | Login      | POST /auth/login              | Login dengan password                       | -                           | email/phone, password                                            | access_token                                                 | Not needed   |
| 2  | Login      | POST /auth/login/biometric    | Login dengan biometric                      | -                           | device_id, signature                                             | access_token                                                 | Not needed   |
| 3  | Register   | POST /auth/register           | Daftar akun baru                            | -                           | full_name, email, phone, password, ktp_no, foto selfie, foto KTP | user_id, verif_status                                        | Not needed   |
| 4  | Dashboard  | GET /loans/active             | Lihat sisa hutang & tagihan bulanan         | Authorization: Bearer token | -                                                                | amount, tenor, sisa hutang, status, daftar tagihan per bulan | Needed       |
| 5  | Apply Loan | POST /loans                   | Ajukan pinjaman baru                        | Authorization: Bearer token | amount, tenor                                                    | loan_id, status                                              | Needed       |
| 6  | Profile    | GET /users/me                 | Lihat data profil & status verifikasi KTP   | Authorization: Bearer token |-                                                                 | full_name, email, phone, verif_status                        | Needed       |
| 7  | Profile    | POST /auth/biometric/register | Daftarkan device untuk bisa login biometric | Authorization: Bearer token | device_id, device_name, public_key                               | status biometrik                                             | Needed       |
